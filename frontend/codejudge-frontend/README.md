# CodeJudge Frontend

This module provides the Vue 3 frontend for CodeJudge.

## Responsibilities

- login and registration pages
- question browsing and filtering
- online code submission
- submission history display
- question creation and editing UI

## Stack

- Vue 3
- TypeScript
- Vue Router
- Vuex
- Arco Design Vue

## Local Run

```bash
npm install
npm run serve
```
